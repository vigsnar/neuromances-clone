PWA Maps¶

Functions for obtaining local affine maps of fully connected neural networks

pwa_maps.lpv_batched(fx, x, use_bias=False)[source]¶
pwa_maps.pwa_batched(fx, x, use_bias=True)[source]¶

function returning parameters of a local affine map of a fully connected neural network for a given input x, where the following holds: y = fx(x) = Astar*x + bstar

Parameters:

  • fx – (nn.Module) fully connected neural network

  • x – (torch.Tensor, shape=[batchsize, in_features])

  • use_bias – flag for turning bias on and off

Returns:

Astar, bstar of a local affine map of the network fx: y = Astar*x + bstar
